
# xpath important queries:

## identifying xpath if no unique attributes present:

- if there is no unique attributes present for child then define xpath for parent/tagNames 
- example
- https://www.google.com/
- it will be following for search bar:
```text
/html/body/div[1]/div[3]/form/div[1]/div[1]/div[1]/div/div[2]/input
```

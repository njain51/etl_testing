
# Page Objects Model

- Page Object Model, also known as POM, is a design pattern in Selenium that creates an object repository for storing all web elements

## Why it's useful?

- Easy Readability 
- Reduce Code Duplication
- Easy to Maintain
- Re-usability of Code
- Reliability
